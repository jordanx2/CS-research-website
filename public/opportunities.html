Useful Embedded Resources for Computer Scientists

Codepen

Codepen is online software tool used for developing HTML, CSS & JavaScript within the browser. Primarily used for showcasing unique and interesting web development elements. Can be viewed as an online code editor where the input and output are both displayed on the same web page.

As can be viewed from the window on the right, this is an example of RGB slider application that outputs the corresponding hexadecimal value. The HTML, CSS & JavaScript code can be all viewed within the window by navigating through the different tabbed panels.

CSS Tricks

As what can probably be interpreted from the name of the site, this platform offers useful insights into the general workings of CSS. Ever since the sites inception from 2007, the platform grew far beyond simply being a site solely for CSS.

It now encompasses the whole basic stack of a simple page i.e., HTML, CSS, JavaScript. Provides extremely useful articles and tips all relating to front-end web development.

Wikipedia

Prehab’s one of the largest and most popular sites on the web today Wikipedia. Wikipedia is an online encyclopaedia. The actual content and articles that is comprised of it is open source in the sense that anyone can edit any Wikipedia page. This often leads to reliability issues when using the platform as a means of research. But all the sources for the content on the actual page can be found in the references section of the page located at the very bottom.

In terms of using Wikipedia for a means of gathering useful information about Computer Science topics, then it’s fairly legit. It gives in dept explanations about keywords relating to CS and a vast range of related topics to the subject. Although there is a stigma around using Wikipedia for general learning and researching purposes due to the unreliability of the site (and rightly so), it’s an extremely valuable tool for quick explanations or to get a general overview of any particular topic.

W3C

World Wide Web Consortium (W3C) is a community formed by members all across the globe. The intend of the W3C is to develop web standards with the help of full time staff and other contributors around the world. The W3C site itself contains everything from the standards that the organization releases, to general facts about W3C as well as some articles.

A useful source for anyone any individual looking to gain a better insight of what goes on under hood in regards to the web. Given that the organization itself is the one that sets standards for the web, this site takes a very comprehensive and thorough in-dept look at all the general aspects of the web.
